# soia-open-media-content-skills

新媒体：写作、封面图片、公众号/X/小红书发布（soia-media-*）

## Domain

This repository owns the `media` domain. Skill names use the `soia-media-*` prefix.

## Install

```bash
npx skills add soia-team/soia-open-media-content-skills -g -a '*' -s <skill> -y
```

## Skills

| Skill | Purpose |
|---|---|
| `soia-media-compose-article-draft` | Turn distilled ideas into a publishable article draft |
| `soia-media-cover-image` | Generate cover images for WeChat, X, and Rednote articles |
| `soia-media-publish-wechat-draft` | Format and upload a WeChat draft without broadcasting it |
| `soia-media-publish-x-thread` | Rewrite an article as an X thread, with optional draft or confirmed publish workflows |
| `soia-media-publish-x-article` | Fill an X Articles draft from Markdown without publishing it |
| `soia-media-publish-rednote-card` | Rewrite an article as Rednote copy for manual publishing |

This repository is part of the SOIA skill ecosystem. The canonical specifications live in [soia-team/soia-open-skills](https://github.com/soia-team/soia-open-skills).
